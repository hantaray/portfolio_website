"Meet"

A serverless, progressive web application (PWA)

Made with React and using a test-driven (TDD) and behaviour driven (BDD) technique. The application uses the Google Calendar API to fetch upcoming events. You can filter events by cities, select the number of events you want to see, show details of events and use the app offline.

React, JavaScript, Jest, Cucumber, OAuth2, AWS Lambda, Recharts, TDD, BDD
